# Phase 1: Monorepo Scaffold

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Working monorepo with both packages wired up to pnpm, Turborepo, Biome, and Changesets — `pnpm turbo build` and `pnpm turbo lint` pass on empty stubs.

**Architecture:** Two packages under `packages/`. Root holds all tooling config. Each package has its own `tsconfig.json` extending a shared base. tsup bundles each package independently; Turborepo caches outputs.

**Tech Stack:** pnpm workspaces, Turborepo, tsup, Biome, Changesets, TypeScript 5.x, Node 22.

## Global Constraints

- `"engines": { "node": ">=22" }` on both packages
- Unscoped npm names: `druckform` and `druckform-mcp`
- Biome for lint/format — no ESLint, no Prettier
- All TypeScript; no JavaScript source files
- `main` branch, squash merges
